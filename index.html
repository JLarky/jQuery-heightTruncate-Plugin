Truncate by Height jQuery Plugin

This is a simple plugin that will truncate a text block based on a given height. By default, calling .widthTruncate() on a selected element will truncate the text block of text down to amount of lines that can fit into height given in css. You can also pass in a pixel value and it will truncate it to that height. See examples below.
